1.This generator is using to create all the json structure for the visa bulletin app.
2.How to use this generator:
 2.1 add the latest visa bulletin data as .xlsx file. The format of the file name is yyyy_MM.xlsx
 2.2 add the file name file_list.txt on the top of file_list.txt
 2.3 run visa_bulletin_generator.py to generator the .json structure.
